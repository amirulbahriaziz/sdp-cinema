# UI Context

> Note: this is a React Native (Expo) app, not web. "Tokens" below live in a shared `theme.ts`
> (a plain JS object) consumed via `StyleSheet`/styled props — there is no CSS. Components must read
> colors/spacing from the theme, never hardcode hex. Final palette/typography is refined in the
> 4.6 UI/UX track; values here are the build baseline derived from the wireframes.

## Theme

Dark only. No light mode. The wireframes are a near-black mobile cinema UI: black backgrounds,
slightly lighter card surfaces, white primary text, muted grey secondary text, pill/outline tags,
and large full-width CTA buttons. Imagery (posters, trailer) sits on dark cards.

## Typography

| Role        | Font                       | Token        |
| ----------- | -------------------------- | ------------ |
| UI text     | System / Inter (Expo font) | `font.sans`  |
| Numeric/mono| System monospace           | `font.mono`  |

Scale: display (movie title), title (screen headers), body, caption (metadata/price). Define as a
`type` map in `theme.ts`.

## Spacing & Radius

| Context              | Token        | Value |
| -------------------- | ------------ | ----- |
| Base unit            | `space.1`    | 4     |
| Inline / chips       | `radius.sm`  | 8     |
| Cards / panels       | `radius.md`  | 12    |
| Modals / sheets      | `radius.lg`  | 20    |

## Component Inventory (build these)

- `Screen` — safe-area dark wrapper.
- `MovieCard` — poster + title + kebab menu (Home carousels).
- `Carousel` / `SectionHeader` ("New Releases ... view all").
- `SearchBar`, `BottomNav` (Home / Tickets / Saved / Profile).
- `Tabs` (Movie Details / Ratings & Reviews; F&B Combo/Food-Snacks/Beverages).
- `RatingStars`, `RatingBreakdown`, `ReviewCard`.
- `Dropdown` (Location, Cinema Hall), `DateStrip`, `TimeSlotChip`.
- `SeatMap` + `SeatCell` (status-driven color), `ScreenCurve`, `SeatLegend`.
- `QtyStepper` (F&B), `FoodItemCard`, `SummaryRow`, `PriceTotalBar`.
- `PaymentMethodRow`, `CardForm`, `PrimaryButton` (full-width CTA), `ResultScreen` (Congratulations).

## Layout Patterns

- Mobile single-column, full-viewport screens; fixed bottom CTA bar (Cancel / Proceed, Book Ticket).
- Home: scroll of horizontal carousels under a greeting + search.
- Detail: hero (trailer/poster) over a dark sheet with tabbed content + sticky bottom CTA.
- Seat screen: legend -> curved "Screen" -> seat grid (row labels both sides) -> sticky subtotal bar.
- Summary/Payment: stacked cards + sticky total + primary action.

## Icons

Expo vector icons (Ionicons/Feather). Stroke-style, consistent sizes (20 inline, 24 nav/buttons).
